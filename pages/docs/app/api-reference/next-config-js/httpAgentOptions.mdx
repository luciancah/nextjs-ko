# httpAgentOptions

Node.js 18 이전 버전에서는 Next.js가 자동으로 [undici](/docs/architecture/supported-browsers#polyfills)로 `fetch()`를 폴리필하고 기본적으로 [HTTP Keep-Alive](https://developer.mozilla.org/docs/Web/HTTP/Headers/Keep-Alive)를 활성화합니다.

서버 측의 모든 `fetch()` 호출에 대해 HTTP Keep-Alive를 비활성화하려면, `next.config.js`를 열고 `httpAgentOptions` 설정을 추가하세요:

```js filename="next.config.js"
module.exports = {
  httpAgentOptions: {
    keepAlive: false,
  },
}
```
