# logging

Next.js를 개발 모드로 실행할 때 데이터 페치가 콘솔에 기록되는 방식을 구성할 수 있습니다.

현재 `logging`은 `fetch` API를 사용한 데이터 페칭에만 적용됩니다. 아직 Next.js 내부의 다른 로그에는 적용되지 않습니다.

```js filename="next.config.js"
module.exports = {
  logging: {
    fetches: {
      fullUrl: true,
    },
  },
};
```

또한, `logging`을 `false`로 설정하여 개발 로깅을 비활성화할 수 있습니다.

```js filename="next.config.js"
module.exports = {
  logging: false,
};
```
