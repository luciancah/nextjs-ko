# typedRoutes (experimental)

정적 타입 링크에 대한 [실험적 지원](/docs/app/building-your-application/configuring/typescript#statically-typed-links). 이 기능을 사용하려면 App Router와 TypeScript를 프로젝트에서 사용해야 합니다.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
};

module.exports = nextConfig;
```
