# Version 10

버전 9에서 10으로의 변경 사항은 없습니다.

버전 10으로 업그레이드하려면 다음 명령어를 실행하세요:

```bash filename="Terminal"
npm i next@10
```

```bash filename="Terminal"
yarn add next@10
```

```bash filename="Terminal"
pnpm up next@10
```

```bash filename="Terminal"
bun add next@10
```

> **참고:** TypeScript를 사용하는 경우, `@types/react` 와 `@types/react-dom` 도 해당 버전으로 업그레이드해야 합니다.
