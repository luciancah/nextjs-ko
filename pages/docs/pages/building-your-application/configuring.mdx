# Configuring

Next.js는 특정 요구 사항을 충족하도록 프로젝트를 사용자 정의할 수 있게 해줍니다. 여기에는 TypeScript, ESlint 등과의 통합뿐만 아니라 절대 경로 가져오기 및 환경 변수와 같은 내부 구성 옵션이 포함됩니다.
