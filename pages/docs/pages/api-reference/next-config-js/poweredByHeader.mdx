# poweredByHeader

기본적으로 Next.js는 `x-powered-by` 헤더를 추가합니다. 이를 비활성화하려면, `next.config.js`를 열고 `poweredByHeader` 설정을 비활성화하세요:

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
