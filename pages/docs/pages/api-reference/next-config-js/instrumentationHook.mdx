# instrumentationHook

실험적인 `instrumentationHook` 옵션을 사용하면 Next.js 앱에서 [`instrumentation` 파일](/docs/app/api-reference/file-conventions/instrumentation)을 통해 계측을 설정할 수 있습니다.

```js filename="next.config.js"
module.exports = {
  experimental: {
    instrumentationHook: true,
  },
}
```
