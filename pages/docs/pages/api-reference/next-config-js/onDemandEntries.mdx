# onDemandEntries

Next.js는 개발 중에 서버가 생성된 페이지를 메모리에 유지하거나 삭제하는 방식을 제어할 수 있는 몇 가지 옵션을 제공합니다.

기본 설정을 변경하려면 `next.config.js`를 열고 `onDemandEntries` 설정을 추가하세요:

```js filename="next.config.js"
module.exports = {
  onDemandEntries: {
    // 서버가 페이지를 버퍼에 유지할 기간 (ms 단위)
    maxInactiveAge: 25 * 1000,
    // 삭제되지 않고 동시에 유지되어야 하는 페이지 수
    pagesBufferLength: 2,
  },
}
```
