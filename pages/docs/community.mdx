# Next.js Community

Next.js는 주간 다운로드 수 500만 건 이상을 기록하며 전 세계적으로 큰 규모의 활발한 개발자 커뮤니티를 보유하고 있습니다. 다음은 우리 커뮤니티에 참여하는 방법입니다:

## Contributing

Next.js 개발에 기여할 수 있는 몇 가지 방법이 있습니다:

- [Documentation](/docs/community/contribution-guide): 개선 사항을 제안하거나 새로운 섹션을 작성하여 사용자들이 Next.js를 사용하는 방법을 이해하도록 도와주세요.
- [Examples](https://github.com/vercel/next.js/tree/canary/contributing/examples): 새로운 예제를 작성하거나 기존 예제를 개선하여 개발자들이 Next.js를 다른 도구 및 서비스와 통합할 수 있도록 도와주세요.
- [Codebase](https://github.com/vercel/next.js/tree/canary/contributing/core): 기본 아키텍처에 대해 더 많이 배우고, 버그 수정, 오류 해결 및 새로운 기능 제안을 통해 기여하세요.

## Discussions

Next.js에 대한 질문이 있거나 다른 사람들을 돕고 싶다면 언제든지 대화에 참여하세요:

- [GitHub Discussions](https://github.com/vercel/next.js/discussions)
- [Discord](https://discord.com/invite/bUG2bvbtHy)
- [Reddit](https://www.reddit.com/r/nextjs)

## Social Media

최신 업데이트를 위해 [Twitter](https://x.com/nextjs)에서 Next.js를 팔로우하고, Next.js 비디오를 위해 [Vercel YouTube 채널](https://www.youtube.com/@VercelHQ)을 구독하세요.

## Code of Conduct

우리는 포용적이고 환영받는 커뮤니티를 만드는 것을 믿습니다. 따라서 모든 회원이 우리의 [행동 강령](https://github.com/vercel/next.js/blob/canary/CODE_OF_CONDUCT.md)을 준수할 것을 요청합니다. 이 문서는 참가자의 행동에 대한 우리의 기대를 설명합니다. 이를 읽고 안전하고 존중받는 환경을 유지하는 데 도움을 주시기 바랍니다.
